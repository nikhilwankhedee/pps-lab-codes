# PPS Lab Codes

This repository contains Python programs developed for the **Programming for Problem Solving (PPS) Laboratory**.

The programs demonstrate fundamental Python concepts including:

* Lists
* Tuples
* Dictionaries
* String operations
* Functions
* Object-Oriented Programming
* NumPy
* Pandas
* Matplotlib

---

# Programs Included

### 1. List Operations

`list_operations.py`

Performs operations on a list of numbers:

* Calculate **sum**
* Calculate **average**
* Display elements in **reverse order**
* Find **minimum and maximum** values

---

### 2. Cricket Score Analysis

`cricket_tuple_analysis.py`

Analyzes cricket scores stored in a tuple:

* Calculate **total runs**
* Find **highest score**
* Count **half-centuries**
* Display scores in **reverse order**

---

### 3. Employee Salary Dictionary

`employee_dictionary.py`

Stores employee details using a dictionary and performs:

* Find **highest salary**
* Find **lowest salary**
* Increase salary of all employees by **10%**
* Display updated employee details

---

### 4. Email Validation Program

`email_validator.py`

Validates and processes an official email ID using string operations:

* Extract **username**
* Extract **domain**
* Perform basic **email validation**

---

### 5. List Processing Function

`list_function.py`

Defines a function that:

* Prints elements **three times if they are numbers**
* Prints elements **with '#' appended if they are not numbers**

Example Input:

```
['23', 'MAN', 'GIRIRAJ', '24', 'ZARA']
```

---

### 6. Student Management using OOP

`student_oop.py`

Uses Object-Oriented Programming to:

* Accept student details
* Store marks using a list
* Calculate **total** and **average**

---

### 7. Data Analysis using NumPy, Pandas and Matplotlib

`numpy_pandas_plot.py`

Creates a dataset of student marks and:

* Converts it into a **Pandas DataFrame**
* Displays data in **tabular format**
* Plots a **line graph** using Matplotlib

---

# Requirements

Install required Python libraries:

```
pip install numpy pandas matplotlib
```

# How to Run

Run any program using:

```
python3 filename.py
```

Example:

```
python3 list_operations.py
```
